## 服务器插件 
sublime server，将下载的插件放到插件包（Sublime Text\Data\Packages）中，然后在工具中找到sublime setting设置如下：
{
    "attempts": 5,
    "autorun": false,
    "defaultExtension": ".html",
    "interval": 500,
    "mimetypes":
    {
        "": "application/octet-stream",
        ".c": "text/plain",
        ".h": "text/plain",
        ".markdown": "text/x-markdown; charset=UTF-8",
        ".md": "text/x-markdown; charset=UTF-8",
        ".py": "text/plain"
    },
    "port": 8080
}
